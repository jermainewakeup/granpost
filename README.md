 Social Media Content Generator + Scheduler using ChatGPT and Facebook Scheduler

## Installation

Clone repo and install dependencies:

```zsh
git clone
cd granpost
pip install -r requirements.txt
```

## Usage

Run the CLI:
```zsh
granpost 
```

Example:
```zsh
granpost -G captions -T christmas
```
